# SKM Service (working title / codename: *Lynx*)

[Documentation on Swagger Hub](https://app.swaggerhub.com/apis/basaas/secret-service/2.1.0/)

### Basic usage

Install packages
```zsh 
yarn
```

Start local lynx
```zsh 
yarn start
```

Watch server and restart after code changes
```zsh 
yarn watch
```

Test lynx components
```zsh 
yarn test
```
